Student Dashboard

Completed Scope:

1. There should be a dashboard and user can add widgets in to the dashboard.
2. User should be able to select two types of widgets.(Summary/Chart)
3. User shoul be able to create summary cards for a selected filter.
4. User should be able to create charts for given two types and user can add filter to selected chart.
5. User can add the same widget multiple times with different filters.
6. Data should be populated in the widget according to the defined filters

Left to do:

1. The user should be able to edit settings of added widgets.

Assumptions Made:

1. The backend for the project is implemented.
2. All the filtering functions are implemented in the backend and, necessary APIs are implemented.
3. For the widgets, user can select one filter to get the data.
